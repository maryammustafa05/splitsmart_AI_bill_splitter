# SplitSmart — AI Bill Splitter

AI-powered bill splitting app that splits expenses and explains them clearly.

🔗 Live Demo: https://splitsmart-ai-bill-splitter.vercel.app/
⚙️ Backend API: https://splitsmartaibillsplitter-production.up.railway.app/docs

## Tech Stack
- FastAPI + Python (backend)
- Groq LLM (AI explanations)
- Vercel (frontend) + Railway (backend)
